Инструкция
  1. Количество вопросов в тесте зависит от разделов, выбранных на начальной форме
  2. Подсказки в тесте зависят от уровня сложности, выбранного на начальной форме (легкий уровень - у каждого вопроса есть варианты ответов; средний уровень - у аждого вопроса указано количество букв в ответе и есть ответ скрытый *, который открывает буквы с течением времени; сложный уровень - подсказки отсутствуют)
  3. На каждый вопрос дается 60с; есть неограниченное количество возможностей ввести правильный овет до истичении времени; при введении правильного ответа, появляется следующий вопрос
  4. Подтвердить ответ можно с помощью копки "ОК" либо клавиши "Enter"
  5. Пропустить вопрос можно с помощью копки "Пропустить" либо клавиши "Esc"
  6. По окончанию теста будет выведен % правильных ответов
  7. Чтоб начать тест, жми на кнопку. Удачи!
